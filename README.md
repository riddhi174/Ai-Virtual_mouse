# ✋ AI Gesture Controller (Virtual Mouse)

AI Gesture Controller is a computer vision-based project that allows users to control their system using hand gestures instead of a physical mouse or keyboard.

## 🚀 Features
- 🖱️ Cursor movement using hand tracking  
- 👆 Left click, right click, and double click  
- ✊ Drag and drop functionality  
- 🔊 Volume control using pinch gesture  
- 💡 Brightness control using pinch gesture  
- 📜 Vertical & horizontal scrolling  
- ✌️ Gesture-based command execution  
- 🎯 Real-time hand tracking with high accuracy  

## 🛠️ Tech Stack
- Python  
- OpenCV  
- MediaPipe  
- PyAutoGUI  
- Pycaw (Volume Control)  
- Screen Brightness Control
